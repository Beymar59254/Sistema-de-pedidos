1) Código HTM / Frontend

Estructura general:
- Carrusel/galería de terrenos (render directo con imágenes de Google Drive).
- Botón “Ver Detalle” por terreno.
- Modal de detalle + botón “Agendar” que:
  1) selecciona el terreno (inputs ocultos anuncioSeleccionado y codigoAnuncio),
  2) hace smooth scroll al formulario.
- Formulario de contacto con submit interceptor.

Inyección de eventos:
- listeners para botones [data-detail-open].
- listeners para submit del form #formVisita.

Envío asíncrono al backend:
fetch(URL_API, { method: 'POST', mode: 'cors', body: JSON.stringify(payload) })

Manejo de respuesta:
- SUCCESS => modal verde + botón a WhatsApp
- CONFLICT => modal roja + botón de coordinación de alternativa
